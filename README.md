## FPM-PLUGIN-MQTT-CLIENT
用于 mqtt-client 的插件

### Install
```bash
npm add fpm-plugin-mqtt-client --save
```

### Useage

- config

  ```javascript
  {
    "mqttserver":{
      "host": "localhost",
      "port": 1883,
      "username": "admin",
      "password": "123123123",
      "clientId": "foo",
    }
  }
  ```

- subscribe

- other
